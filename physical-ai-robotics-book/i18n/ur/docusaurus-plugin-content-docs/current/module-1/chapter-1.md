

# ماڈیول 1: روبوٹک اعصابی نظام (ROS 2)

## باب 1: ROS 2 کا تعارف

### ذیلی باب 1: ROS 2 کے بنیادی اصولوں کو سمجھنا

#### نوڈز (Nodes)

**تعارف**
یہ سیکشن ROS 2 میں نوڈز کے تصور کو متعارف کراتا ہے، جو بنیادی پروسیسز ہیں جو کمپیوٹیشن کرتے ہیں۔

**تفصیل**
ROS 2 میں، ایک نوڈ ایک قابلِ عمل پروسیس ہے جو دیگر نوڈز کے ساتھ مواصلت کے لیے ROS 2 کلائنٹ لائبریریز کا استعمال کرتا ہے۔ نوڈز ٹاپکس، سروسز، اور ایکشنز کا استعمال کرتے ہوئے دوسرے نوڈز سے پیغامات وصول اور بھیج سکتے ہیں۔ یہ ROS 2 سسٹم کے بنیادی اجزاء ہیں، جو ماڈیولر اور تقسیم شدہ روبوٹ سافٹ ویئر کی ترقی کو ممکن بناتے ہیں۔ مثال کے طور پر، ایک کیمرہ ڈرائیور ایک نوڈ ہو سکتا ہے جو امیجز شائع کرتا ہے، اور ایک نوڈ جو ان امیجز پر کارروائی کرتا ہے وہ کیمرے کے ٹاپک کو سبسکرائب کر سکتا ہے۔

**کوڈ مثال**
```python
# Python میں بنیادی ROS 2 نوڈ مثال
import rclpy
from rclpy.node import Node

class MyNode(Node):
    def __init__(self):
        super().__init__('my_minimal_node')
        self.get_logger().info('Hello from my minimal ROS 2 node!')

def main(args=None):
    rclpy.init(args=args)
    minimal_node = MyNode()
    try:
        rclpy.spin(minimal_node)
    except KeyboardInterrupt:
        pass
    finally:
        minimal_node.destroy_node()
        rclpy.shutdown()

if __name__ == '__main__':
    main()
```

**کوئز**
ROS 2 نوڈ کا بنیادی کردار کیا ہے؟
a) روبوٹ کے ہارڈویئر کا انتظام کرنا۔
b) کمپیوٹیشن کرنا اور دیگر نوڈز کے ساتھ مواصلت کرنا۔
c) روبوٹ کے کنفیگریشن فائلز کو محفوظ کرنا۔
d) روبوٹ کنٹرول کے لیے صارف انٹرفیس فراہم کرنا۔

**لغت**
-   **Node**: ROS 2 میں ایک قابلِ عمل پروسیس جو کمپیوٹیشن کرتا ہے اور دیگر نوڈز کے ساتھ مواصلت کرتا ہے۔
-   **rclpy**: ROS 2 کے لیے Python کلائنٹ لائبریری۔

ROS 2 نوڈز ROS 1 کے نوڈز سے ایک ارتقاء ہیں، جو بہتر سیکیورٹی، حقیقی وقت کی صلاحیتوں، اور DDS (Data Distribution Service) کے لیے معاونت کے ساتھ ڈیزائن کیے گئے ہیں۔ ماڈیولر ایگزیکیوٹیبلز جو مڈل وئیر پر مواصلت کرتے ہیں کا تصور مرکزی حیثیت رکھتا ہے۔
